### Commit Convention
### 기본 형식

```swift
[prefix] #이슈번호 - 이슈 내용
```


### Prefix의 의미

```bash
[Feat]: 새로운 기능 구현
[Fix]: 버그, 오류 해결, 코드 수정
[Add]: Feat 이외의 부수적인 코드 추가, 라이브러리 추가, 새로운 View 생성
[Del]: 쓸모없는 코드, 주석 삭제
[Refactor]: 전면 수정이 있을 때 사용합니다
[Remove]: 파일 삭제
[Chore]: 그 이외의 잡일/ 버전 코드 수정, 패키지 구조 변경, 파일 이동, 파일이름 변경
[Docs]: README나 WIKI 등의 문서 개정
[Merge]: #이슈번호 - Merge
```

### Git Branch
브랜치 
```
feature/#이슈번호

fix/#이슈번호

[Merge] #이슈번호 - Merge
```
